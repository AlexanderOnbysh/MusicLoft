# Music Loft

## How to run

```bash
# Create virtualenv
python3 -m venv .env

# Activate environment
source ./.env/bin/activate

# Run server
python manage.py runserver
```

## Description
Django project for web development course.

Mutual project with [Alexander Onbysh](https://github.com/AlexanderOnbysh)
